calculate Perps Funding Fee Calculator

Funding rate data source: Binance and Bitget (from 2025-02-18T16:00:00+08:00)

Funding fee expenses can vary significantly across different exchanges when trading the same asset using perpetual contracts.
This web app is designed to calculate and compare funding fee costs or income over a specified period on Bitget and Binance.

Github: https://github.com/btxLithium/perps-funding-fee-calculator
